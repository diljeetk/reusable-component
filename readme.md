# Reusable Components

## Packages

* Header
* Styles
* Dropdown

## Setup
* `npm install -g yarn`
* `git clone <repo url>`
* `yarn install`
* `yarn packageInstall`

## Publish Changes
* Run `lerna bootstrap`
* Run `lerna publish`

## How to publish new component
* First commit your new component with git repo using `git commit -am "msg"`
* Run `lerna bootstrap`
* Run `lerna publish`
